لوحة التحكم — إضافة/تحديث عرض

أضف عطرًا جديدًا أو اربط عرضًا (متجر + أحجام/أسعار) بعطر موجود بالفعل — سيتم التعرف عليه تلقائيًا بالاسم.

⚙️ إعداد الاتصال بـ GitHub (مرة واحدة)

أنشئ التوكن من GitHub → Settings → Developer settings → Fine-grained tokens، وامنحه صلاحية Contents: Read and write على هذا المستودع فقط. يُحفظ في متصفحك أنت فقط ولا يُرسل لأي جهة غير GitHub.

✏️ تعديل:

بيانات العطر

المكونات (Fragrantica) — اختياري، اتركه فارغًا للإبقاء على المكونات الحالية

المتجر وعروضه (اختياري عند تعديل الاسم فقط — اتركه فارغًا لعدم تغيير المتاجر)

بعد النشر، GitHub Pages يعيد بناء الموقع خلال دقيقة تقريبًا.
